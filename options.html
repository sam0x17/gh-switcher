    Pattern Syntax

    • github.com/org-name — matches the org page exactly
    • github.com/org-name/* — matches one path segment (e.g. any repo under the org)
    • github.com/org-name/** — matches anything under that org (repos, issues, PRs, etc.)

    Tip: Place more specific patterns above general ones. The first matching pattern wins.
